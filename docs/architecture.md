## Security Architecture

| Layer | Protection |
|-------|-----------|
| Code | ESLint rules, SonarQube analysis |
| Dependencies | npm audit, Trivy filesystem scan |
| Container | Multi-stage build, non-root user, Trivy image scan |
| Network | Kubernetes NetworkPolicies, Helmet headers |
| Secrets | GitHub Secrets, K8s Secrets (never in code) |
| Runtime | Resource limits, health probes, auto-restart |

---

## Data Flow

1. **User** sends HTTP request to the API
2. **Kubernetes Service** routes to an available pod
3. **Express middleware** processes (security â†’ logging â†’ metrics)
4. **Route handler** executes business logic
5. **Response** sent back through middleware chain
6. **Metrics** collected by Prometheus on next scrape
7. **Grafana** visualizes metrics in dashboards
8. **AlertManager** fires alerts if thresholds are breached
